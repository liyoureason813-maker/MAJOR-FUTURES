# MAJOR FUTURES V3.5｜无登录公开版

此版本已移除家长登记、登录状态、退出按钮及 Google Sheet 相关代码。

## GitHub Pages 部署

将本文件夹中的以下文件上传到 GitHub 仓库根目录并覆盖旧文件：

- `index.html`
- `styles.css`
- `app.js`
- `matcher.js`
- `README.md`

然后确认：

- Settings → Pages
- Source：Deploy from a branch
- Branch：main
- Folder：/(root)

网站地址保持不变。更新后可在网址末尾加 `?v=35` 并按 `Ctrl + Shift + R` 强制刷新。
